为了避免在服务端加载、渲染客户端组件，我们可以：

- 使用 SPA 渲染，请参考 <Link href="/render-modes" />
- 使用 SSR 渲染，同时仅在浏览器端加载渲染组件，请参考 <Link href="/dynamic-import" />

> **客户端组件**
>
> 有一些组件库只能在浏览器中使用，
> 例如 [react-leaflet](https://github.com/PaulLeCam/react-leaflet) 或 [vue-leaflet](https://github.com/vue-leaflet/vue-leaflet)
>
> 许多客户端组件支持在服务端渲染（服务端渲染空壳，浏览器端实际渲染）
>
> 但有一些客户端组件不支持服务端渲染，甚至可能会造成程序崩溃
