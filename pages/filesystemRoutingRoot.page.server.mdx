`_default.page.route.js` 的 `export { filesystemRoutingRoot }` 允许我们更改文件系统路由的 URL 根目录。

> **不要使用 `filesystemRoutingRoot` 来更改 base URL**，请参考 <Link href="/base-url" />

> `filesystemRoutingRoot` 通常用于 [域名驱动的文件系统](/file-structure#domain-driven).

## Example

```js
// /marketing/_default.page.route.js

// 我们将 marketing 页面路由到 `/*` 而不是 `/marketing/*`
export const filesystemRoutingRoot = '/'
```

```
/marketing/_default.page.route.js
/marketing/pages/index/index.page.js
/marketing/pages/about/index.page.js
/marketing/pages/jobs/index.page.js
```

结果：

```
PROJECT FILES                                    URL
/marketing/pages/index/index.page.js             /
/marketing/pages/about/index.page.js             /about
/marketing/pages/jobs/index.page.js              /jobs
```
