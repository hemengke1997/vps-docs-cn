环境：`Browser`、`Node.js`

`.page.js` 文件可以导出以下内容：

- `export { Page }`（或 `export default`）
- `export { onBeforeRender }`
- <Link href='/exports' />

`.page.js` 文件是同构的：它既可以在浏览器中运行，也可以在 Node.js 中运行

`.page.js` 文件是懒加载的：它只在需要时加载，这意味着如果没有 URL 请求匹配页面的路由，那么 `.page.js` 永远不会加载到浏览器中，也不会加载到 Node.js 进程的内存中
