`onBeforeRoute()` hook 提供了对路由的完全控制能力

它通常用于 [国际化](/i18n).

我们也可以用它来[实现自定义路由器](https://github.com/brillout/vite-plugin-ssr/issues/140)

<Note type="warning">对于 i18n 以外的用例，请打开 GitHub ticket 或联系 Discord：`onBeforeRoute()` 是有意限制的，但可以解除限制以解锁新功能。</Note>
